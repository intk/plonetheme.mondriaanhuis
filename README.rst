============
 Mondriaanhuis theme
============
------------------------------
 An installable theme for Plone 5
------------------------------

`Latest version on PyPI <https://pypi.python.org/pypi/plonetheme.mondriaanhuis>`_

Documentation
=============
- Based on the Plone 5 theme `Clean Blog <https://pypi.python.org/pypi/plonetheme.clean-blog>`_

Installation
============================
Install plonetheme.mondriaanhuis by adding it to your buildout::

    [buildout]

    ...

    eggs =
        plonetheme.mondriaanhuis

and then running ``bin/buildout``
